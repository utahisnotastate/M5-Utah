# Utah Flux Studio

**Build M5Stack hardware like Lego — no command line required.**

Utah Flux Studio is the main application in this repository. It combines:

- **Utah-Flux** — visual brick library and project compiler
- **m5resolver** — intent engine, safety gates, and agentic recovery
- **Firmware** — one-time flash, then runtime updates via JSON intents

Double-click `Start Utah Flux Studio.bat` (Windows) or run `utah-flux-studio` after `pip install -e host`.
