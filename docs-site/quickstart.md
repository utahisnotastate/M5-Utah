# Quickstart

## For everyone (no CLI)

1. Run **Install Utah Flux.bat** once (installs Python package).
2. Double-click **Start Utah Flux Studio.bat**.
3. In the browser:
   - Pick a **Starter** template (e.g. Hello Screen)
   - Click **Connect Device** (USB + Chrome/Edge)
   - Click **Play Project**
4. Tilt or interact with the device to see results.

## One-time firmware (adult)

Flash `firmware/` once with PlatformIO so the device understands Utah Flux intents.

After that, users only need the visual studio.

## Save your work

Use **Save Project** to download a `.flux.json` file. Open it later with **Open Project**.
