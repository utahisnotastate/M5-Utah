    非技术用户教程 — 上课或办工作坊

    对象： 家长、老师、活动组织者
    目标： 不教编程也能带完整一节课

    课前准备（约 30 分钟，成人）

    1. 安装 Python 3.10+
    2. 运行 Install Utah Flux.bat
    3. 在 firmware/ 烧录一次固件
    4. 测试：打开 Studio → Hello Screen → 连接 → Play

    课中（约 60 分钟）

    A 部分 — 演示（10 分钟）

    • “我们用积木，不用写代码。”
    • 演示：倾斜 → 文字 + 声音，连线，Play

    B 部分 — 动手（35 分钟）

    1. 每人打开 Utah Flux Studio
    2. 选模板：入门 Hello / 活动 Tilt Alarm / 欢乐 Party
    3. 改文字和颜色
    4. Save Project 保存

    C 部分 — 展示（15 分钟）

    学生演示作品，讨论“一个触发连多个动作”。

    日常用法

    任务 操作
    打开 双击 Start Utah Flux Studio.bat
    加载 Open Project
    运行 Connect Device → Play
    停止 Stop
    分享 复制 .flux.json 文件

    不需要命令行

    所有操作都在 Utah Flux Studio 里完成。
