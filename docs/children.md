# Kids Guide / 儿童指南

## 1) What is this? / 这是什么？

**EN**  
Your M5Stack is like a tiny robot brain.  
This project gives it one “super language” so it can listen and react fast.

**中文**  
你的 M5Stack 就像一个小小的机器人大脑。  
这个项目给它一种“超级语言”，让它更快地听懂并做出反应。

## 2) What can you do? / 你可以做什么？

**EN**
- show messages on screen
- make beeps and tones
- react when you tilt the device
- build mini games and experiments

**中文**
- 在屏幕显示文字
- 发出提示音
- 在你倾斜设备时做出反应
- 做小游戏和小实验

## 3) First experiment / 第一个实验

**EN**
Ask an adult to run:
- `python examples/tilt_tone.py --port COM3`

Then:
1. tilt left/right
2. watch numbers change
3. hear pitch go up/down

**中文**
请大人帮你运行：
- `python examples/tilt_tone.py --port COM3`

然后：
1. 左右倾斜设备
2. 看数字变化
3. 听声音变高或变低

## 4) Fun challenge ideas / 有趣挑战

**EN**
- make a “balance game” with score text
- map tilt to different colors
- create a “danger alarm” when movement is too fast

**中文**
- 做一个“平衡游戏”并显示分数
- 用倾斜角度控制颜色变化
- 当动作太快时触发“警报”

## 5) Safety / 安全提醒

**EN**
- always use help from an adult for wiring and flashing firmware
- keep liquids away from electronics
- unplug before touching wires

**中文**
- 接线和烧录固件请在大人帮助下进行
- 电子设备远离液体
- 接线前先断电

## 6) Big idea / 核心想法

**EN**  
Old way: many different code libraries to remember.  
New way: one project, one style, many cool creations.

**中文**  
旧方法：要记很多不同代码库。  
新方法：一个项目，一种风格，做出很多酷作品。
