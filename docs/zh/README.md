## 功能（v0.3+）

- 可视化乐高积木 IDE（用户无需命令行）
- Utah-Flux 编译器（`.flux.json` 项目）
- 意图驱动控制（`display`、`speaker`、`power`、`registry`）
- 安全校验与仿真门禁
- Agent 遥测自修复
- 硬件上下文协议（`registry/units.json`）
- 入门模板与保存/打开项目
