# 儿童教程 — 搭建第一个设备项目

**时间：** 约 15 分钟  
**需要：** 大人帮忙接 USB 和第一次烧录固件  
**不需要：** 输入命令

## 第 1 步 — 打开 Utah Flux Studio

请大人帮你：

1. 运行一次 `Install Utah Flux.bat`
2. 双击 `Start Utah Flux Studio.bat`

浏览器会打开彩色应用。

## 第 2 步 — 试试“Hello Screen”

1. 在左边 **Starters** 点击 **👋 Hello Screen**
2. 画布上会出现两个连好的积木
3. 用 USB 连接 M5Stack
4. 点击 **🔌 Connect Device**（用 Chrome 或 Edge）
5. 点击 **▶ Play Project**

屏幕上应显示 **Hello!**

## 第 3 步 — 自己做

1. 拖 **When Device Tilts** 到画布
2. 拖 **Show Message** 到下面
3. 点倾斜积木右边的白点，再点消息积木左边的白点
4. 点消息积木，在右边把文字改成你的名字
5. 点 **Play**，倾斜设备！

## 第 4 步 — 加声音

1. 拖 **Play Sound** 到画布
2. 把倾斜积木也连到声音积木
3. 在右边选 **high** 音调
4. 再 Play，边倾斜边听！

## 第 5 步 — 保存作品

点 **💾 Save Project** 保存。下次用 **📂 Open Project** 打开。

## 积木速查

| 积木 | 作用 |
|------|------|
| 🚀 When Project Starts | 按 Play 时运行 |
| 📐 When Device Tilts | 倾斜时触发 |
| 💬 Show Message | 屏幕文字 |
| 🎵 Play Sound | 发声 |
| 🎉 Party Mode | 欢乐模式 |
| 🛡️ Safe Mode | 保护设备 |

## 出问题怎么办

- 不能连接？用 Chrome 或 Edge
- 屏幕没反应？请大人检查 USB 和固件
- 看右边 **Live Log**，绿色好，红色要找大人

你像搭乐高一样完成了硬件项目，真棒！
