# 儿童指南

Utah Flux Studio 让你像搭 **乐高** 一样做 M5Stack 项目，**不用输入命令**。

## 快速步骤

1. 双击 **Start Utah Flux Studio**
2. 拖积木到画布
3. 连接白点
4. **Connect Device** → **Play Project**
