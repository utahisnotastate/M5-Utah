# 儿童指南

## 用 Utah Flux Studio 搭建

你**不需要**输入命令。

1. 请大人双击 **Start Utah Flux Studio**。
2. 从左边拖积木到大画布上。
3. 用白点把积木连起来（像乐高一样）。
4. 用 USB 连接 M5Stack。
5. 点击 **Connect Device**，再点 **Play Project**。

## 试试模板

点击左边的入门模板，比如 **Hello Screen** 或 **Party Mode**，然后按 Play。

## 保存作品

点击 **Save Project** 把项目保存到电脑。

## 安全

- 接线和第一次烧录固件请在大人帮助下进行。
- 电子设备远离液体。
