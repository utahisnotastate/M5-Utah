# 非技术用户指南

## Utah Flux Studio 能做什么

- 一个可视化应用完成教学与演示
- 无需命令行和代码编辑器
- 用 `.flux.json` 保存和分享项目
- 内置入门模板

## 日常使用

1. 双击 **Start Utah Flux Studio.bat**
2. 打开项目或选模板
3. USB 连接设备
4. **Connect Device** → **Play Project**
5. 结束点 **Stop**
