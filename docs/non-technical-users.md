# Non-Technical User Guide / 非技术用户指南

## 1) What this gives you / 你能得到什么

**EN**  
This project helps your M5Stack device behave consistently without needing many separate apps and repositories.  
Think of it as:
- one stable device firmware
- one control app on your computer
- one shared catalog of supported modules

**中文**  
这个项目让你的 M5Stack 设备更一致地工作，不再依赖大量分散仓库与应用。  
你可以把它理解为：
- 一个稳定的设备固件
- 一个电脑上的统一控制程序
- 一个共享的模块支持目录

## 2) What you need / 需要准备什么

**EN**
- an M5Stack device (CoreS3 recommended)
- USB cable
- a computer
- basic install help from your team

**中文**
- 一台 M5Stack 设备（推荐 CoreS3）
- USB 数据线
- 一台电脑
- 团队提供的基础安装帮助

## 3) First run steps / 首次运行步骤

**EN**
1. Ask your technical teammate to flash the firmware once.
2. Install the control software on your computer.
3. Start the example command.
4. Move or tilt the device and observe text/sound changes.

**中文**
1. 请技术同事先帮你完成一次固件烧录。
2. 在电脑安装控制软件。
3. 启动示例命令。
4. 移动或倾斜设备，观察屏幕和声音变化。

## 4) Daily workflow / 日常使用流程

**EN**
- Connect device
- Start control script
- Run your workflow
- Stop script and unplug

No need to recompile firmware for every small logic change.

**中文**
- 连接设备
- 启动控制脚本
- 执行业务流程
- 停止脚本并断开设备

不需要每次小逻辑改动都重新编译固件。

## 5) Why this is simpler than many repositories / 为什么比多个仓库简单

**EN**
- fewer moving parts
- fewer places to search
- less confusion about “which library to use”
- faster support from your team

**中文**
- 组件更少
- 查找位置更集中
- 不再纠结“该用哪个库”
- 团队支持更高效

## 6) FAQ / 常见问题

**EN**
- **Do I need to code?** Not for normal usage.
- **Can I break the device by trying scripts?** Usually no; scripts can be restarted quickly.
- **Can I still use old tools?** Yes, but this project should be your default path for new work.

**中文**
- **我必须会写代码吗？** 日常使用不需要。
- **尝试脚本会把设备弄坏吗？** 一般不会；脚本可快速重启恢复。
- **还能用旧工具吗？** 可以，但新工作建议默认使用本项目。
