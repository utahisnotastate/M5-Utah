# Non-Technical User Guide

## What Utah Flux Studio gives you

- One visual app for all M5Stack lessons and demos
- No command line, no code editor required
- Save and share projects as `.flux.json` files
- Built-in starters for instant success

## How you work day to day

1. Double-click **Start Utah Flux Studio.bat**
2. Open an existing project or pick a starter
3. Connect the device with USB
4. Click **Connect Device**, then **Play Project**
5. Click **Stop** when finished

## Full tutorial

See [Non-Technical Tutorial](tutorials/non-technical-tutorial.md) for running a class or workshop.

## Why this replaces many old repos

| Old pain | Utah Flux answer |
|----------|------------------|
| Which library do I install? | Drag a brick |
| Recompile and flash every change | Press Play |
| Hard to see what device does | Open the project file |
